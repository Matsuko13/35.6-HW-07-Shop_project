# Shop project

```Запуск проекта```

1) Скачать репозиторий на ПК (ссылка);
2) Установить необходимые расшинения:
      - В основной дериктории Shop_project использовала ```npm install```;
      - Далее в дерикториях Shop.Admin и Shop.Api  так же использовала```npm install```;
      - В дериктории Shop.Client использовала ```yarn install```;
      - Собрала данное приложение из дериктория Shop.Client с помощью ```yarn run build```.
3) Запуск проекта происходит из основного дериктория Shop_project с помощью команды ```nodemon```
4) Проект будет доступен по локальному хосту 3000 (http://localhost:3000).
